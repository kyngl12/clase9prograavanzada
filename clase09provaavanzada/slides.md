# ¿Qué es Fortnite Battle Royale?

Fortnite es un videojuego de battle royale desarrollado por **Epic Games** que combina:

- 🎯 **Acción intensa** - Combate en tiempo real contra 99 jugadores
- 🏗️ **Construcción única** - Edifica estructuras para defenderte o atacar
- 🎨 **Estilo colorido** - Gráficos vibrantes y dinámicos
- 👥 **Modo multijugador** - Solo, dúo, equipo o Squad (hasta 4 jugadores)
- 💰 **Modelo gratuito** - Free-to-play con compras cosméticas
- 🌍 **Comunidad masiva** - Millones de jugadores en todo el mundo

<br>

> **Lanzado en:** 25 de julio de 2018  
> **Disponible en:** PC, PS4, Xbox One, Nintendo Switch, Mobile y Cloud Gaming

# 🎮 Gameplay

## Mecánicas principales

| Mecánica | Descripción |
|----------|-------------|
| **Salto desde el autobús** | Comienza cayendo desde el cielo |
| **Recolecta de recursos** | Destruye estructuras para obtener madera, piedra y metal |
| **Construcción** | Construye muros, rampas y estructuras defensivas |
| **Combate** | Múltiples armas: rifles, escopetas, explosivos |
| **Movimiento rápido** | Sprints, saltos y parkour para movilidad |
| **Zona de seguridad** | Mapa se reduce gradualmente, forzando encuentros |

## Objetivo Final

**Ser el último jugador o equipo en pie**

🏆 ¡Victory Royale! 🏆

# 🎯 Modos de Juego

Fortnite ofrece diversos modos:

# 💰 Sistema de Batalla Pass

El Battle Pass es el sistema de progresión premium de Fortnite.

## Niveles de Acceso

```
GRATUITO (Free Track)
- 10 recompensas básicas
- Puntos de experiencia
- Primeros cosmetics

PREMIUM (950 V-Bucks)
- 100 recompensas exclusivas
- Skins legendarias
- Mochilas y herramientas
- Emotes y sprays
```

## Progresión
- Gana niveles completando misiones
- Sube de nivel jugando en cualquier modo
- Acceso a Batallas de Élite para más recompensas

# 👥 Skins y Cosmética

La cosmética de Fortnite es una de sus principales características:

## Tipos de Recompensas

- **Skins** 👤 - Personajes únicos y exclusivos
- **Mochilas** 🎒 - Accesorios de espalda personalizados
- **Herramientas** 🔨 - Herramientas de cosecha temáticas
- **Planeadores** 🪂 - Modelos para paracaídas
- **Emotes** 💃 - Bailes y animaciones
- **Sprays** 🖌️ - Graffitis para marcar superficies
- **Emojis** 😊 - Iconos para expresar emociones

# 🗺️ El Mapa

El mapa de Fortnite es dinámico y cambia con cada temporada.

## Características del Mapa

- **Tamaño:** 2x2 km de área jugable
- **Rotación:** Cambia con cada temporada
- **Puntos de Interés:** Ciudades, bases militares, lugares famosos
- **Vehículos:** Autos, motos y lanchas para movilidad rápida
- **Dinámico:** Cambios de clima, eventos y catástrofes

## Biomas Típicos

```
Urbano • Rural • Desierto • Nevado • Tropical
```
